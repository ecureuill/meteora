## About

Meteora is the result of the Alura's Front-end Challenge, developed based on the provided Figma design. 

## Running Locally

To run the Adopet App locally, follow these steps:

```bash 
# Clone this repository
git clone https://github.com/ecureuill/meteora.git
# Install the dependencies
npm install
# Start the development server
npm start
```
Access the app in your web browser at: http://localhost:3000
